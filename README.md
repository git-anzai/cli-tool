### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `tzld` 命令了。

- `tzld init vue-template myVue`
- `tzld config get`
- `tzld config set type orgs`
- `tzld config set registry vuejs-templates`

- `tzld config set type users`
- `tzld config set registry xxx`

### 发布

开发完成后，即可发布至 npm.
